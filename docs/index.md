## 🎯 快速开始

1. 阅读 [部署文档](Deployment) 完成 Docker 部署
2. 登录管理后台 `http://你的IP:10307`（Emby 管理员账号）
3. 进入 [系统设置](Configuration) 填写核心配置
4. 把用户中心 `http://你的IP:10308` 分享给用户
