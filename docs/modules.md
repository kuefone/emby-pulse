# 🎬 功能模块说明

> 本页对 EmbyPulse 核心模块进行详细说明，包含使用方法与配置路径。

---

## 🌐 端口与访问说明

EmbyPulse 启动后会同时运行 **两个独立端口**（物理隔离）：

| 端口 | 用途 | 访问地址 | 登录方式 |
| :---: | --- | --- | --- |
| `10307` | 🔒 管理员后台 | `http://你的IP:10307` | Emby 管理员账号 |
| `10308` | 👤 用户中心（求片/个人） | `http://你的IP:10308` | Emby 普通用户账号 |

> ⚠️ **安全提示**：10308 端口是物理隔离的独立引擎，只允许访问求片页面、登录页和静态资源，无法越权进入管理后台。

<details>
<summary>📌 如何把用户中心链接分享给用户</summary>

1. 确认你的服务器公网 IP 或域名
2. 用户中心访问地址为：`http://你的域名或IP:10308`
3. 将此链接发给用户即可，用户使用自己的 Emby 账号密码登录
4. 如果你做了反向代理，可以把 10308 端口映射到自定义域名（如 `https://request.example.com`）

</details>

## 📊 1. 全景仪表盘（Dashboard）

> 🔗 访问：`http://你的IP:10307` → 登录后默认进入

**功能**：实时播放监控 · 资源总量统计 · 天/周/月趋势图

<details>
<summary>📖 使用方法</summary>

1. 登录后自动进入仪表盘
2. 顶部查看实时播放会话（客户端、IP、转码状态）
3. 中部展示资源总量与播放统计
4. 底部趋势图可切换"天 / 周 / 月"维度
5. 点击用户排行榜可查看更多详情

</details>

| 设置位置 | 配置项 |
| --- | --- |
| 系统设置 → 媒体服务器核心 | `emby_host` / `emby_api_key` / `playback_data_mode` |
| 系统设置 → 隐藏黑名单 | `hidden_users` |

## 🔍 2. 缺集管理（Gap Management）

> 🔗 访问：管理后台 → 左侧菜单"缺集管理"

**功能**：TMDB 比对扫描 · MoviePilot 联动补集 · 下载器截胡精准补集

<details>
<summary>📖 使用方法</summary>

1. 进入"缺集管理"页面
2. 点击"扫描缺集"并等待结果
3. 选择缺集条目，点击进入"基因配型中心"
4. 系统自动匹配 4K / 1080P / HDR 等版本
5. 选择资源后点击"提交并自动截胡"

</details>

| 设置位置 | 配置项 |
| --- | --- |
| 系统设置 → MoviePilot 自动化联控 | `moviepilot_url` / `moviepilot_token` / `pulse_url` |
| 缺集管理页面 → 底层接管配置弹窗 | `client_type` / `client_url` / `client_user` / `client_pass` |

## 🎬 4. 求片系统（Request Center）

> 🔗 用户端：`http://你的IP:10308` · 管理端：后台 → "求片管理"

**功能**：用户提交求片 · 管理员审核流转 · 入库自动闭环 · 机器人通知

<details>
<summary>📖 用户端使用方法</summary>

1. 打开 `http://你的IP:10308`，使用 Emby 账号密码登录
2. 在"全站大厅"搜索想看的影片
3. 点击提交求片请求
4. 在"个人中心"查看请求状态

</details>

<details>
<summary>📖 管理端使用方法</summary>

1. 登录管理后台 `http://你的IP:10307`
2. 进入"求片管理"页面
3. 审核、下发或拒绝请求
4. 入库完成后系统自动更新状态并通知用户

</details>

| 设置位置 | 配置项 |
| --- | --- |
| 系统设置 → 对外服务门户 | `emby_public_url` / `client_download_url` / `welcome_message` |
| 通知与机器人 → Telegram 通道 | `tg_bot_token` / `tg_chat_id` |

## 👥 5. 用户管理（User Center）

> 🔗 访问：管理后台 → 左侧菜单"用户管理"

**功能**：用户资料管理 · 续期/禁用/重置 · 批量操作 · 邀请系统

<details>
<summary>📖 使用方法</summary>

1. 管理员登录后台，点击左侧"用户管理"
2. 点击某个用户的"配置"按钮查看详情
3. 在弹窗中可切换"基础资料 / 库权限 / 高级策略"三个 Tab
4. 修改到期时间、备注、权限后点击"保存配置"
5. 勾选多个用户可使用底部"批量续期 / 批量套用"操作栏

</details>

<details>
<summary>🎟️ 邀请系统使用</summary>

1. 点击右上角"生成邀请"按钮
2. 选择有效期（7 天 / 30 天 / 永久）
3. 复制生成的邀请链接发给新用户
4. 新用户打开链接即可自助注册（无需管理员干预）
5. 点击"邀请管理"可查看所有邀请码的使用状态

</details>

| 设置位置 | 配置项 |
| --- | --- |
| 系统设置 → 媒体服务器核心 | `emby_host` / `emby_api_key` |

## 🤖 8. Telegram 机器人

> 🔗 配置：管理后台 → "通知与机器人"页面

**功能**：播放推送 · 入库海报 · 命令交互 · 黑名单拦截告警

<details>
<summary>📖 配置步骤</summary>

1. 在 Telegram 中找到 `@BotFather`，发送 `/newbot` 创建机器人
2. 获取 Bot Token
3. 获取你的 Chat ID（可通过 `@userinfobot` 获取）
4. 登录管理后台 → 通知与机器人页面
5. 在"推送场景总控"中开启"启用底层引擎"
6. 在"Telegram 通道"中填入 `Bot Token` 和 `Admin Chat ID`
7. 点击"发送测试"验证连通性
8. 点击"保存配置"

</details>

**可用命令**

| 命令 | 说明 |
| --- | --- |
| `/search 关键词` | 搜索 Emby 媒体库 |
| `/stats` | 查看服务器统计数据 |
| `/check` | 系统健康诊断 |

## 🧹 14. 去重管理（Dedupe）

> 🔗 访问：管理后台 → 左侧菜单"去重管理"

**功能**：智能扫描重复文件 · 画质/大小对比 · 白名单保护 · 释放存储空间

<details>
<summary>📖 使用方法</summary>

1. 进入"去重管理"页面
2. 点击"智能全库扫描"开始扫描
3. 扫描完成后切换"电影 / 剧集"Tab 查看重复项
4. 对比每组重复文件的画质、大小等信息
5. 选择需要删除的副本进行清理
6. 点击"白名单"可保护不想被扫描的内容
7. 点击"扫描配置"可调整扫描规则

</details>
